# number-click-experiment-system

本プロジェクトは、大学修士課程（M1）の研究のために開発しましたが、  
**研究計画の変更により、実際の実験では使用されていません。**

PC（Web ブラウザ）上で実施する実験システムとして設計されており、  
被験者は Web ブラウザ上で課題を行い、課題中には  
**時刻情報および数字列が一定のルールで提示される。**

本システムは、  
**提示された数字列を順にクリックしていくタスク**と  
**時刻情報の提示**を組み合わせた実験を想定して開発されています。

## 構成
- `client/` : 被験者向け Web UI（数字クリック課題・時刻／数字列の提示など）
- `server/` : PHP バックエンド  
  （※本リポジトリには **サンプル実装のみ** を含み、実運用コードやデータは含まれていません）
